# Sistema de Loja Virtual Simplificada
Este é um mini projeto da disciplina de Programação Orientada a Objetos (POO), ministrada pelo professor Jayr Alencar no curso de Engenharia de Software, da Universidade Federal do Cariri (UFCA). O objetivo é implementar um sistema de loja virtual simples, utilizando Python, POO, persistência simples e uma interface CLI ou API mínima.

# Objetivo do Projeto
Construir um sistema capaz de gerenciar as principais operações de uma loja virtual:

- Cadastro e manutenção de produtos e clientes

- Manipulação de carrinho de compras

- Criação e controle de pedidos

- Registro de pagamentos

- Cálculo de frete

- Aplicação opcional de cupons de desconto

- Emissão de resumo/nota do pedido

- Geração de relatórios de vendas

# Classes Principais
- `Produto`
- `Clientes`
- `Endereço`
- `Carrinho`
- `ItemCarrinho`
- `Cupons`
- `Pedido`
- `ItemPedido`
- `Pagamento`
- `Frete`
